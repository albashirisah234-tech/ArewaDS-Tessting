# ArewaDS-Tessting

A beginner-friendly learning repository for **Python** and **Machine Learning**.

## What this repository contains

- Python fundamentals examples
- Introductory machine learning examples
- Unit tests to validate sample implementations

## Repository structure

```text
learning_repository/
  __init__.py
  python_basics.py
  machine_learning.py
tests/
  test_learning_repository.py
requirements.txt
```

## Getting started

1. Create and activate a virtual environment.
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Run tests:

   ```bash
   python -m unittest discover -s tests -v
   ```

## Learning path

1. Start with `learning_repository/python_basics.py`
2. Continue with `learning_repository/machine_learning.py`
3. Validate understanding by reading and running `tests/test_learning_repository.py`
